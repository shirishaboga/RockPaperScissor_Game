# Rock-Paper-Scissor Game (Java)

## Description
This is a simple Rock-Paper-Scissor game developed in Java.  
It allows the user to play against the computer for 3 rounds and keeps track of scores.

## Features
- 3-round game against the computer
- Menu-driven input for Rock, Paper, or Scissor
- Displays round results and final winner
- User-friendly interface with input validation

## Language Used
Java

## How to Run
1. Compile the program using a Java compiler (`javac RockPaperScissor_Game.java`)
2. Run the program (`java RockPaperScissor_Game`)
3. Follow on-screen instructions to choose Rock, Paper, or Scissor
4. Check the round results and final winner
